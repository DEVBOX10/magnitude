---
title: 'Windsurf MCP'
description: 'Enable Windsurf to build and run Magnitude tests with MCP'
---

1. Install MCP server via npm:
```
npm i -g magnitude-mcp
```
2. Add this to your `./codeium/windsurf/model_config.json`:
```json
{
  "mcpServers": {
    "magnitude": {
      "command": "npx",
      "args": [
        "magnitude-mcp"
      ]
    }
  }
}
```
